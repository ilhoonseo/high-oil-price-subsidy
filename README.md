# ⛽ 고유가 피해 지원금 수령 현황판

대상자 37명의 고유가피해지원금 수령 여부를 아름답고 직관적으로 체크 및 관리하며, 별도의 데이터베이스 없이도 타인에게 수령 여부 상태를 동기화하여 보여줄 수 있는 초경량 단일 페이지 웹 애플리케이션(SPA)입니다.

---

## ✨ 주요 기능
- **프리미엄 UI/UX**: 세련된 미드나잇 다크 모드와 글래스모피즘(Glassmorphism) 효과를 통해 현대적이고 고급스러운 첫인상을 제공합니다.
- **실시간 요약 통계**: 총 대상자 수, 수령 완료자 수, 미수령자 수 및 전체 지급률을 실시간으로 집계하고 아름다운 프로그레스 바와 함께 표현합니다.
- **검색 & 필터링**: 성함으로 실시간 대상자를 검색할 수 있으며, 탭을 통해 '전체', '수령 완료', '미수령' 대상자만 필터링해서 볼 수 있습니다.
- **오작동 방지 안전장치**: **[관리자 편집 모드]** 토글이 제공됩니다. 이 토글이 켜진 상태에서만 수령 상태(체크박스)를 편집할 수 있으며, 꺼진 상태에서는 읽기 전용 모드로 동작하여 의도치 않은 클릭 실수를 미연에 방지합니다.
- **데이터베이스가 필요 없는 상태 공유 (URL 압축 인코딩)**: 
  - 37명의 상태 값을 37비트의 2진 코드로 결합한 뒤, 이를 10자리의 16진수 문자열로 압축합니다.
  - 관리자가 체크를 완료하고 **[공유 링크 생성]**을 누르면 고유 식별 주소(예: `?state=3a7f2d01b0`)가 생성 및 복사됩니다.
  - 이 주소를 공유받은 상대방은 데이터베이스 연동 없이도 관리자가 조율해 둔 상태 그대로 현황판을 보게 됩니다.
- **로컬 스토리지 자동 백업**: **[로컬에 저장]** 버튼을 통해 현재 수정 상황을 기기의 브라우저에 저장하여, 다음에 다시 방문할 때도 기존 작업을 이어서 진행할 수 있습니다.

---

## 🛠️ 기술 스택
- **HTML5** & **Vanilla CSS3** (다크 테마, CSS 변수 시스템, HSL 컬러 팔레트, 반응형 웹 디자인)
- **Vanilla JS** (ES6+, Bitwise Hex 인코딩/디코딩, 돔 핸들링)
- **Lucide Icons** (세련되고 경량화된 오픈소스 아이콘 팩)

## 🚀 사용법 및 배포
본 프로젝트는 완전한 정적 웹 페이지이므로 GitHub Pages, Vercel, Netlify 등을 통해 10초 만에 무료 배포가 가능합니다.

### 로컬 실행 방법
1. 본 레포지토리를 다운로드(Clone)합니다.
2. `index.html` 파일을 더블 클릭하여 즉시 브라우저에서 실행할 수 있습니다.

### 배포 후 상태 관리 방법
1. 웹 사이트에 접속합니다.
2. **관리자 편집 모드** 스위치를 켭니다.
3. 지원금을 지급/수령한 대상자 옆의 체크박스를 조율합니다.
4. **[공유 링크 생성]** 버튼을 누릅니다.
5. 복사된 주소를 단톡방이나 안내를 보낼 채널에 공유합니다.
6. 사람들은 해당 링크를 누르기만 하면 실시간으로 업데이트된 현황을 확인할 수 있습니다!
